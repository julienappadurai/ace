Google Drive Setup

To sync with Drive, you need a free Google Cloud OAuth Client ID. Create one at console.cloud.google.com under APIs & Services → Credentials → OAuth 2.0 Client ID (Web Application). Add your GitHub Pages URL as an authorised origin.
